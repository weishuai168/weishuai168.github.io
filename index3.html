CF-Blog主题开发文档-史上最简单博客主题开发

发布时间：2021-01-03 分类：CF-Blog相关 阅读：5517

由于今天的版本更新，改用模板引擎mustache，主题开发变得非常简单，主题文件只有2个

本博客程序，主题开发只需要两个文件

1. index.html (列表页模板)

2. post.html (文章页模板)

模板语法

使用mustache模板引擎，语法非常简单，比如输出标题：

{{title}}

循环文章列表：

{{#articles}} <h2><a href="{{url}}">{{title}}</a></h2> <p>{{description}}</p> {{/articles}}

可用变量

列表页可用变量：

{ "title": "网站标题", "articles": [ { "title": "文章标题", "url": "文章链接", "description": "文章摘要", "date": "发布日期", "view": "阅读量", "img": "缩略图" } ], "pagination": { "next": "下一页链接" } }

文章页可用变量：

{ "title": "文章标题", "content": "文章内容", "date": "发布日期", "view": "阅读量", "category": "分类", "tags": ["标签1", "标签2"], "prev": "上一篇链接", "next": "下一篇链接" }

主题存放路径

在网站根目录的theme文件夹下，新建一个文件夹（比如mytheme），然后把index.html和post.html放进去，在后台切换主题即可。
